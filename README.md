qrcode
======

# chrome二维码插件

## 使用方法
* 将源码下载以后，在chrome的右上角菜单中选择，“工具” -> “扩展程序” , 然后点击上的“加载正在开发的扩展程序”,选择下载下来的源代码目录就可以用了

* 直接用crx文件安装
